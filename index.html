Projects

Smart City: IoT-based Earthquake and Wind Speed Detection System with Website Integration

The purpose of this device is to monitor the surrounding conditions and detect early signs of natural disasters. This project integrates three sensors into a single tool, including a vibration sensor, temperature sensor, and anemometer. Once the data is collected, it will be displayed on a screen and also presented through a website.

Smart Farm IoT Device with 3 Sensor Integration

This device have some features sensors for humidity, temperature, total dissolved solids (TDS), and water pH. The data collected from these sensors is transmitted to a website using the Blynk platform therefore allowing it to be accessed by others, even from remote locations. Additionally, the device is equipped with a connector port that can be used to activate a water pump, enabling automatic irrigation.
